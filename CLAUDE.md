# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with content in this repository.

## 目录结构

- `raw_material/` — 存放《世说新语》原文及相关材料。
- `lecture/` — 存放生成的讲座文稿。
- `CLAUDE.md` — 本文件。

## 生成讲座

1. **风格**：以顾随《讲文选》的风格，解说《世说新语》的内容。
2. **篇幅**：5000 字以上。
3. **输入**：用户每次会贴一些文件、或一些《世说新语》原文作为材料。
4. **生成**：根据这些材料写一篇新的讲座，保存到 `lecture/`。

**上下文连贯**：写新讲座时，先看看 `lecture/` 里已有的篇目，注意与之前内容的呼应与连贯，必要时可以参考、回扣前作。
